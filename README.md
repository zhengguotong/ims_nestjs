# Invoicing Management  System - Nestjs

## Description

[VisionSoft](https://www.visionsoft.com.au/) Invoicing management System (IMS) -  [Nest](https://github.com/nestjs/nest) version 

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
